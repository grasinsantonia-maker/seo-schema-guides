# SEO Schema Guides

A collection of structured data (Schema.org) implementation guides optimized for SEO and GEO (Generative Engine Optimization).

## What's Included

| Schema Type | File | Description |
|-------------|------|-------------|
| Organization | `ORGANIZATION_SCHEMA_GUIDE.md` | Complete guide for Organization/LocalBusiness schema |

## Purpose

These guides are optimized for:
- **Google Search** — Rich results, Knowledge Panels
- **AI Search Engines** — ChatGPT, Perplexity, Gemini, Google AI Overview
- **Entity Recognition** — Building semantic footprint

## Quick Start

1. Read the relevant schema guide
2. Gather required information (see checklist in each guide)
3. Generate schema following the template
4. Validate at https://search.google.com/test/rich-results
5. Implement in website `<head>` section
